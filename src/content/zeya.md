# 一、项目背景

## 1.1 国家政策驱动

2024年5月，国家数据局发布《深化智慧城市发展推进城市全域数字化转型的指导意见》，首次提出“探索发展数字低空基础设施”，为城市全域数字化转型指明方向。2024年12月，水利部发布《数字孪生水利“天空地水工”一体化监测感知夯基提能行动方案（2024-2026年）》，旨在加速构建全要素、全天候、动态监控的综合体系。

## 1.2 浙江省及温州市政策导向

温州市积极响应国家战略，于2026年1月发布《温州市支持低空经济高质量发展的若干措施》，明确支持公共服务领域低空应用，鼓励各部门在生态治理、城市管理等场景先行先试。

根据《温州市低空基础设施布局和空域航路航线规划（征求意见稿）》，温州将建设7个通用机场、超1000个垂直起降点，规划579条低空航线，构建覆盖全域的低空交通网络。

浙江省积极推动水利数字化改革，温州市大力推进水库现代化运行管理矩阵建设。2025年8月，省水利厅专家组赴泽雅公司调研指导水库现代化运行管理矩阵试点建设工作，强调深化系统贯通与功能落地、健全多库群与跨区域协同机制、加快形成可推广的样板经验。

泽雅水库作为市区重要饮用水源地，其智能化巡检体系建设正当其时。

## 1.3 泽雅水库的战略地位

泽雅水库是温州市重要的饮用水源地，集雨面积102平方公里，总库容5794万立方米，年供水量11360万立方米，承担温州市区约20%的供水量，被誉为温州的“生命线”之一。1999年，温州市政府划定泽雅水库饮用水源保护区范围，依法进行保护。2021年，瓯海区对水源保护区进行了优化调整，一级保护区面积为3.53km²，二级保护区面积为19.32km²，准保护区面积为75.76km²。

# 二、业务需求分析

## 2.1 项目痛点

- 效率瓶颈：水域面积1.6平方公里，岸线长约20公里，人工巡检日均覆盖≤5公里，效率低下；

- 覆盖盲区：库区周边地形复杂，部分岸线难以到达，存在监控死角；

- 时效性差：发现问题依赖人工上报，响应链条长，难以实现即时预警；

- 数据孤岛：现有监控系统与水质监测数据尚未完全整合，缺乏统一分析平台。

## 2.2 业务需求

#### 2.2.1 客户类型与需求痛点

| 客户类型 | 需求 | 痛点 |
|---|---|---|
| 水库管理单位（泽雅公司） | 实现库区全要素智能感知，提升巡检效率 | 人力投入大，覆盖不全，数据整合难 |
| 生态环境部门 | 实时监控水质变化，及时发现污染源 | 传统监测点位有限，难以溯源 |
| 水利主管部门 | 掌握大坝、溢洪道等设施安全状况 | 人工巡检频次低，隐患发现滞后 |
| 应急管理部门 | 突发事件快速响应，辅助指挥决策 | 缺乏空中视角，现场态势难掌握 |

#### 2.2.2 业务需求分析

**水质与水环境监测**

- 水质参数监测：实时检测水温、pH值、溶解氧、浊度等指标，分析水质变化趋势

- 水面状况观察：查看水面垃圾、漂浮物、藻类爆发等，评估水体清洁度

- 污染源排查：监测入库支流（西山溪、石桥溪、龙坑溪）及周边污染源影响

- 隔离防护设施检查：一级保护区环库线约2.5公里隔离防护设施完好性检查

**水利设施安全监测**

- 大坝安全：检查坝体表面裂缝、渗漏、滑坡等情况

- 取水口及泵站：监测取水口周边状况、泵站设施运行状态

- 泄洪设施：检查溢洪道、泄洪闸等设施完好性

**安全与应急管理**

- 库区安全：巡查非法捕鱼、游泳、倾倒垃圾等行为

- 地质灾害隐患：观察周边山体滑坡、崩塌迹象

- 应急响应：突发水污染、洪水等事件时快速获取现场数据

- 防汛指挥：汛期空中态势感知，辅助防汛调度

**数据采集**

- 采集巡检过程中的图像、视频和各类传感器数据，进行分析处理，生成巡检报告和数据分析图表，直观展示水库状况和变化趋势。

# 三、项目目标

## 3.1 总体目标

通过无人机自动化巡检系统建设，实现泽雅水库“天空地水工”一体化监测感知，构建全要素、全天候、动态监控的综合体系，提升水库智能化管理能力

## 3.2 具体目标

1. 提速增效：自动化作业替代人工巡检，实现库区全覆盖监测，巡检频率提升5倍以上

2. 实时监测：5G网络实时传输数据，AI自动分析识别异常，异常发现时间缩短至30分钟内

3. 应急响应：突发事件无人机快速部署，15分钟内抵达现场，实时回传影像数据

4. 数据智能化管理：构建全生命周期数字档案、分类分级预警和水库综合健康评价体系

5. 精细监测：坝体巡检覆盖率100%，及时发现裂缝、渗漏等安全隐患

6. 水源地保护：实时监控一级保护区，杜绝非法活动对水源的污染

# 四、建设与服务内容

## 4.1 整体架构

借鉴深圳水库、佛子岭水库等先进经验，泽雅水库无人机巡检系统采用“无人机+自动化机巢+云平台”架构，形成“感知-传输-处理-决策”闭环：

**感知层—数据采集**

- 无人机数据采集系统：多旋翼无人机平台+多类型任务挂载；

- 自动化机巢系统：无人机自动起降、充电、存储、数据传输；

**传输层—数据传输**

- 通信传输系统：4G/5G+专用无线通信，确保实时数据传输稳定；

**平台层—数据处理与管理**

- 云平台处理系统：数据存储、AI分析、报告生成（大疆司空2/第三方云平台）；

**决策层—业务应用**

- 监控指挥中心/移动终端：任务规划、调度监控、决策支持。

### 5.3.5 综合ROI结论

**核心结论：** 泽雅水库无人机巡检项目不是"成本节约型"投资，而是"价值提升型"投资。

| 直接成本节约 | ★★☆☆☆（每年节省约 1.8 万元，回收期较长） |
|---|---|
| 效率提升价值 | ★★★★★（巡检效率提升10倍，覆盖范围扩大3倍） |
| 安全风险降低 | ★★★★★（人员零风险、隐患早发现、应急快响应） |
| 管理水平升级 | ★★★★☆（从"人防"到"技防+人防"的质的飞跃） |
| 综合投入产出 | ★★★★☆（综合价值远高于直接成本节约） |

建议切入点

1. 申请数字化转型补贴：水利部、省水利厅均有数字化改革相关补贴政策，可申请专项经费覆盖30-50%建设成本

2. 分期建设：先投入基础版（约 28 万）验证效果，再逐步升级AI识别等高级功能

3. 区域共享：可与周边水库、乡镇共享使用，分摊成本

# 六、预期成果

**效率提升**

- 巡检效率提升：自动化作业替代人工，巡检效率提升5-8倍

- 全覆盖监测：实现水库水域、岸线、大坝、入库支流全覆盖监测

- 频次增加：日常巡检每天1次，特殊时期加密监测

**数据精准度**

- 高精度数据：4800万像素高清影像+热成像，精准捕捉隐患

- 实时传输：5G网络实时回传，异常30分钟内可响应

- AI智能识别：自动识别人员入侵、水面漂浮物等20余类风险

**应急能力**

- 快速部署：突发事件15分钟内抵达现场

- 实时指挥：直播画面辅助远程指挥调度

- 防汛支撑：汛期空中态势感知，提升防洪调度能力

**成本优化**

- 人力成本降低：减少人工巡检投入，降低一线人员作业风险

**管理智能化**

- 全生命周期档案：建立大坝、水质、设施全生命周期数字档案

- 分类分级预警：风险智能预警，支持科学决策

- 综合健康评价：多维度评估水库健康状态

**水源保护成效**

- 一级保护区实时监控：杜绝非法钓鱼、游泳、倾倒垃圾等行为

- 污染源快速发现：及时发现并处置排污口

- 水质持续改善：支持保持或提升现有Ⅲ类水质标准

### 示范意义

本项目建成后，将成为温州市首个“天空地水工”一体化监测感知水库示范点，为温州市及浙江省其他水库的智能化管理提供可复制、可推广的样板经验。
